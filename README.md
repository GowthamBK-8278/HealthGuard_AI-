# ⚕️ HealthGuard AI+
### Your Personal AI-Powered Clinical Assistant

HealthGuard AI+ is a professional-grade medical platform designed to provide instant health insights, diagnostic predictions, and clinical analysis using the power of **Google Gemini AI**. From analyzing symptoms to summarizing lab reports, HealthGuard AI+ brings a doctor-level intelligence directly to your browser.

---

## 🚀 Core Features

### 1. Smart Symptom Analyzer 🧠
Predict conditions based on symptoms with a high-accuracy clinical matching engine.

### 2. AI Health Assistant 🤖
A 24/7 intelligent chatbot powered by Gemini for instant answers to any medical or wellness queries.

### 3. Medical Report Analyzer 📄 [NEW]
Upload images of blood tests, lab reports, or prescriptions to get a plain-English clinical summary.

### 4. AI Diet & Nutrition Planner 🥗 [NEW]
Personalized 1-day meal plans tailored to your age, weight, and specific health goals (Weight Loss, Muscle Gain, etc.).

### 5. Health Analysis Radar Graph 📊 [NEW]
Dynamic visualization of your clinical profile, severity, and diagnostic confidence after every analysis.

### 6. Emergency SOS Locator 🚨
One-click access to find the nearest hospitals using real-time geolocation.

### 7. Nearby Pharmacies Locator 💊 [NEW]
Instantly find the closest pharmacies around you for quick medicine access.

### 8. BMI & Health Risk Calculator ⚖️ [NEW]
Track your body metrics and receive instant AI-driven health risk assessments.

### 9. Dengue Risk Detection 🦟
Specialized logic to detect symptoms matching high-risk diseases like Dengue and suggest immediate action.

### 10. Specialist & Medicine Advice 👨‍⚕️
Receive suggestions for generic medicines and the exact type of specialist you should visit for your condition.

## 🛠️ Technology Stack
- **Backend:** Python / Flask
- **AI Engine:** Google Gemini Pro & Flash (Generative AI)
- **Frontend:** HTML5, Vanilla CSS, Javascript
- **Visuals:** Chart.js (Data Visualization), Inter & Roboto Fonts
- **APIs:** Google Maps (Geolocation), Gemini API

## ⚙️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/GowthamBK-8278/HealthGuard_AI-.git
   cd HealthGuard_AI-
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root directory and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the application:**
   ```bash
   python app.py
   ```
   Access the app at `http://127.0.0.1:5000`
